# Data Processing Agreement Templates

## Payroll Processor
### Data Processing Agreement
- **Parties:** [Your Company Name] ("Controller") and [Payroll Processor Name] ("Processor")  
- **Data Processing:** Processor shall only process personal data as instructed by Controller.  
- **Security Measures:** Implement appropriate technical and organizational measures to ensure a level of security appropriate to the risk.  
- **Sub-Processor Authorization:** Processor shall not engage another processor without prior written consent of Controller.  
- **Data Subject Rights:** Process personal data in compliance with data subjects' rights.  
- **Termination:** Upon termination of this Agreement, Processor shall return or delete personal data as instructed by Controller.
